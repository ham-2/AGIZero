# AGIZero
Chess engine
(https://lichess.org/@/TheAGI/)
(https://naagicode.com)

Started as AGI
which stands for nothing

requires PEXT (BMI2)!

## Build using Cmake
tested on Windows with MSVC and Ubuntu with GCC.

### MSVC
```
$ cmake .
$ cmake --build . --config Release
```

### GCC
```
$ cmake .
$ cmake --build .
```
